## 🎯 Doel per fase

### Fase 1 — HTML + CSS
Leren hoe webpagina's zijn opgebouwd en gestyled.  
**Resultaat:** 3 pagina's live op GitHub Pages.

### Fase 2 — JavaScript
Interactie toevoegen: knoppen, formulieren, animaties.  
**Resultaat:** Werkende interactieve features op de portfolio-pagina's.

### Fase 3 — PHP + MySQL + Linux
Leren hoe data van een formulier naar een server gaat en in een database wordt opgeslagen.  
**Resultaat:** Een werkend CRUD-systeem (bijv. takenlijst of gastenboek).

### Fase 4 — Eindproject
Alles samenvoegen in één zelfstandig project, zonder tutorials.  
**Resultaat:** Een complete full-stack applicatie als portfoliostuk.
